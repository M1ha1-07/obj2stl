Privacy & Security Model

Last updated: June 27, 2026

1. Zero-Data Architecture

This utility operates on a strict Client-Side Processing model. We do not rent, own, or operate backend file servers. When you select a 3D model, the file is never transmitted over the internet.

The conversion from OBJ to Binary STL happens entirely inside your device's active memory (RAM) using standard WebGL and JavaScript execution.

2. Analytics & Tracking

We believe utilities should be quiet. This site does not use Google Analytics, Meta Pixels, or any third-party behavioral trackers. We do not know who you are, where you are from, or what you are converting.

3. Cookies & Local Storage

This application does not set any advertising or tracking cookies. If we release future premium features (like batch-processing), a single localized key will be stored in your browser's localStorage to verify your unlock status. This key never leaves your machine.

Verify it yourself

Don't trust, verify. You can view the entire source code for this pipeline to confirm there are no network requests being made during conversion. You can also disconnect your computer from the Wi-Fi before clicking convert—the tool will still work flawlessly.
